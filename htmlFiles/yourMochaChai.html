Now, it's time to write your own tests!

First, check out the requirements these functions need to accomplish.

Second, find the yourMochaChai_test.js to write your own tests.

Then, head to yourMochaChai.js to write your own functions.

function createUser

write a function called createUser that accepts a name, PIN, and initial account balance, and saves that in a user object, returning the entire object. It should:

  • Only allow 4 digit PIN numbers, throwing an error if PINS are not the right length
  • Have the option to randomly generate a PIN
  • Throw an error if a user doesn't open the account with any funds

function sortUsers

write a function called sortUsers that sorts all users from lowest to highest amount of money in their account. It should

  • Throw an error if any user has an undefined amount
  • Throw an error if it receives something that's not an array of users
  • Actually organize and order users
